## 문서 생성기 (Documentation Generator)

Forge 문서 생성기의 동작과 관련된 구성입니다. 이 키들은 `[doc]` 섹션에 설정됩니다.

##### `out`

- Type: string
- Default: `docs`
- Environment: `FOUNDRY_DOC_OUT`

생성된 문서의 출력 경로입니다.

##### `title`

- Type: string
- Environment: `FOUNDRY_DOC_TITLE`

생성된 문서의 제목입니다.

##### `book`

- Type: string
- Default: `./book.toml`
- Environment: `FOUNDRY_DOC_BOOK`

사용자가 제공한 `book.toml`의 경로입니다. 문서 생성 중에 기본 설정과 병합됩니다.

##### `repository`

- Type: string
- Environment: `FOUNDRY_DOC_REPOSITORY`

git 저장소 URL입니다. git 소스 파일에 대한 링크를 제공하는 데 사용됩니다.
누락된 경우, `forge`는 현재 origin url을 조회하여 그 값을 사용하려고 시도합니다.

##### `ignore`

- Type: array of strings (patterns)
- Default: `[]`
- Environment: `FOUNDRY_DOC_IGNORE`

문서를 생성할 때 무시할 파일 목록입니다. 이는 glob 패턴의 쉼표로 구분된 목록입니다.
