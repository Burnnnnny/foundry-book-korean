## Etherscan

API 키와 같은 Etherscan 관련 구성입니다. 이 구성은 Forge의 여러 곳에서 사용됩니다.

`[etherscan]` 섹션은 키와 Etherscan 구성 테이블의 매핑입니다.

Etherscan API V2에서는 Etherscan 키만 유효하며, 이를 사용하여 모든 유사한 탐색기(예: BscScan / BaseScan / Polygonscan)에 액세스할 수 있습니다.

```toml
[etherscan]
mainnet = { key = "${ETHERSCAN_API_KEY}" }
mainnet2 = { key = "ABCDEFG", chain = "mainnet" }
optimism = { key = "1234567" }
unknown_chain = { key = "ABCDEFG", url = "<etherscan api url for this chain>" }
```
