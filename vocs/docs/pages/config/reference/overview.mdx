## 구성 개요 (Config Overview)

Foundry의 구성 시스템을 통해 도구를 구성할 수 있습니다.

### 프로필 (Profiles)

구성은 프로필로 임의의 네임스페이스를 지정할 수 있습니다. 기본 프로필의 이름은 `default`이며, 다른 모든 프로필은 이 프로필의 값을 상속합니다. 프로필은 `profile` 맵에 정의됩니다.

`local`이라는 프로필을 추가하려면 다음과 같이 추가합니다:

```toml
[profile.local]
```

`FOUNDRY_PROFILE` 환경 변수를 설정하여 사용할 프로필을 선택할 수 있습니다.

### 전역 구성 (Global configuration)

`~/.foundry` 폴더에 `foundry.toml` 파일을 생성하여 Foundry를 전역적으로 구성할 수 있습니다.

### 환경 변수 (Environment variables)

구성은 `FOUNDRY_` 및 `DAPP_` 접두사가 붙은 환경 변수로 재정의할 수 있습니다.

예외는 다음과 같습니다:

- `FOUNDRY_FFI`, `DAPP_FFI`, `DAPP_TEST_FFI`
- `FOUNDRY_PROFILE`
- `FOUNDRY_REMAPPINGS`, `DAPP_REMAPPINGS`
- `FOUNDRY_LIBRARIES`, `DAPP_LIBRARIES`
- `FOUNDRY_FS_PERMISSIONS`, `DAPP_FS_PERMISSIONS`, `DAPP_TEST_FS_PERMISSIONS`
- `DAPP_TEST_CACHE`
- `DAPP_TEST_FUZZ_RUNS`
- `DAPP_TEST_FUZZ_DEPTH`

### 구성 형식 (Configuration format)

구성 파일은 [TOML 형식](https://toml.io)으로 작성되며, 섹션 내에 간단한 키-값 쌍이 있습니다.

이 페이지에서는 각 구성 키를 자세히 설명합니다. 기본값을 보려면 이 문서의 특정 키를 참조하거나 [기본 구성](/config/reference/default-config)을 참조하세요.

### 구성 키 (Configuration keys)

이 섹션에서는 모든 구성 키를 문서화합니다. 모든 구성 키는 `default`와 같은 프로필 아래에 있어야 합니다.
