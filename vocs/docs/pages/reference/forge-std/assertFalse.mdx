## `assertFalse`

### 시그니처

```solidity
function assertFalse(bool data) internal;
```

```solidity
function assertFalse(bool data, string memory err) internal;
```

### 설명

`data`가 거짓(false)이라고 주장(assert)합니다.

선택적으로 리버트(revert) 문자열에 오류 메시지를 포함할 수 있습니다.

### 예제

```solidity
bool failure = contract.fun();
assertFalse(failure);
```
