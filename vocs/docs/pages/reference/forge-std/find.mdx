## `find`

function find(StdStorage storage self) internal returns (uint256);

주어진 [`target`](/reference/forge-std/target.mdx), [`sig`](/reference/forge-std/sig.mdx), [`with_key`](/reference/forge-std/with_key.mdx)(s), [`depth`](/reference/forge-std/depth.mdx)를 사용하여 임의의 스토리지 슬롯을 찾습니다.
