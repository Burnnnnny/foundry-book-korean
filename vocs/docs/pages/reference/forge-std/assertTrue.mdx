## `assertTrue`

### Signature

```solidity
function assertTrue(bool data) internal;
```

```solidity
function assertTrue(bool data, string memory err) internal;
```

### Description

`data`가 참(true)임을 단언합니다.

선택적으로 revert 문자열에 에러 메시지를 포함할 수 있습니다.

### Examples

```solidity
bool success = contract.fun();
assertTrue(success);
```
