## `deal`

### Signature

```solidity
function deal(address to, uint256 give) public;
```

```solidity
function deal(address token, address to, uint256 give) public;
```

```solidity
function deal(address token, address to, uint256 give, bool adjust) public;
```

### Description

[`deal`](/reference/cheatcodes/deal.mdx) 치트코드를 래핑한 것으로, 대부분의 ERC-20 토큰에서도 작동합니다.

`deal`의 대체 시그니처가 사용되면 잔액을 설정한 후 토큰의 총 공급량(total supply)을 조정합니다.

### Examples

```solidity
deal(address(dai), alice, 10000e18);
assertEq(dai.balanceOf(alice), 10000e18);
```
