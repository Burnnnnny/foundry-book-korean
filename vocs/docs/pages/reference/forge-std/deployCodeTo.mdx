## `deployCodeTo`

### Signature

```solidity
function deployCodeTo(string memory what, address where) internal virtual;
```

```solidity
function deployCodeTo(string memory what, bytes memory args, address where) internal virtual;
```

```solidity
function deployCodeTo(string memory what, bytes memory args, uint256 value, address where) internal virtual;
```

### Description

아티팩트 디렉토리에서 컨트랙트 바이트코드를 가져와 임의의 주소에 컨트랙트를 의사 배포(pseudo-deploy)합니다. 이는 프로덕션 환경을 재현하는 데 사용할 수 있습니다.

calldata 파라미터는 `ContractFile.sol`(파일명과 컨트랙트 이름이 같은 경우), `ContractFile.sol:ContractName`, 또는 프로젝트 루트에 상대적인 아티팩트 경로 형식이 될 수 있습니다.

`value` 파라미터를 사용하여 값을 전달할 수도 있습니다. 이는 생성 시 ETH를 전송해야 하는 경우 필요합니다.

### Examples

```solidity
deployCodeTo("MyContract.sol", abi.encode(arg1, arg2), arbitraryAddr);
```
