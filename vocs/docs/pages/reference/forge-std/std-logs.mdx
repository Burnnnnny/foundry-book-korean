## Std 로그 (Std Logs)

Std 로그는 [`DSTest`](../ds-test.md#logging) 라이브러리의 로깅 이벤트를 확장합니다.

### 이벤트 (Events)

```solidity
event log_array(uint256[] val);
event log_array(int256[] val);
event log_named_array(string key, uint256[] val);
event log_named_array(string key, int256[] val);
```

### 사용법

이 섹션에서는 사용 예시를 제공합니다.

#### log\_array

```solidity
event log_array(<type>[] val);
```

여기서 `<type>`은 `int256`, `uint256`, `address`일 수 있습니다.

##### 예시

```solidity
// 스토리지를 가정
// uint256[] data = [10, 20, 30, 40, 50]; 

emit log_array(data);
```

#### log\_named\_array

```solidity
event log_named_array(string key, <type>[] val);
```

여기서 `<type>`은 `int256`, `uint256`, `address`일 수 있습니다.

##### 예시

```solidity
// 스토리지를 가정
// uint256[] data = [10, 20, 30, 40, 50]; 

emit log_named_array("Data", data);
```
