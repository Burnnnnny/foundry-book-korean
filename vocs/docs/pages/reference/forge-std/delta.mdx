## `delta`

### Signature

```solidity
function delta(uint256 a, uint256 b) internal pure returns (uint256)
```

```solidity
function delta(int256 a, int256 b) internal pure returns (uint256)
```

### Description

두 숫자의 차이를 절대값으로 반환합니다.

### Example

```solidity
uint256 four = stdMath.delta(-1, 3);
```
