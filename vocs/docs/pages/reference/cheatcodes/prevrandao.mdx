## `prevrandao`

### 시그니처

```solidity
function prevrandao(bytes32) external;
```

### 설명

`block.prevrandao`를 설정합니다.

Paris 하드 포크 이전의 EVM 버전에서 사용하면 되돌려집니다(revert). 이 경우 대신 [`vm.difficulty`][prevrandao]를 사용하세요.

### 예제

```solidity
vm.prevrandao(bytes32(uint256(42)));
emit log_uint(block.prevrandao); // 42
```
