## `startStateDiffRecording`

### 시그니처

```solidity
function startStateDiffRecording()
```

### 설명

호출 컨텍스트와 함께 CREATE, CALL 또는 SELFDESTRUCT 작업 코드(opcode)의 일부로 모든 상태 변경을 순서대로 기록합니다.
기록된 상태 변경에 액세스하고 해석하는 방법에 대한 자세한 내용은 [`stopAndReturnStateDiff`](/reference/cheatcodes/stop-and-return-state-diff.mdx)를 참조하세요.
