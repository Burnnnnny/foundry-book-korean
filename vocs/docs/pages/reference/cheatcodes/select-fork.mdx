## `selectFork`

### 시그니처

```solidity
function selectFork(uint256 forkId) external;
```

### 설명

`createFork`에 의해 생성된 포크 식별자를 받아 해당 포크 상태를 활성 상태로 설정합니다.

### 예제

이전에 생성된 포크 선택:

```solidity
uint256 forkId = vm.createFork(MAINNET_RPC_URL);

vm.selectFork(forkId);

assertEq(vm.activeFork(), forkId);
```
