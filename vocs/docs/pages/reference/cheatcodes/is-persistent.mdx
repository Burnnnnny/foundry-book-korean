## `isPersistent`

### 시그니처

```solidity
function isPersistent(address) external returns (bool);
```

### 설명

계정이 영구적([`makePersistent`](/reference/cheatcodes/make-persistent.mdx))으로 표시되었는지 여부를 반환합니다.

### 예제

`msg.sender`와 현재 테스트 계정의 기본 상태 확인

```solidity
// 기본적으로 `sender`와 테스트 컨트랙트 자체는 영구적입니다.
assert(cheats.isPersistent(msg.sender));
assert(cheats.isPersistent(address(this)));
```
