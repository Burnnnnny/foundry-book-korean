## `prank`

### 시그니처

```solidity
function prank(address msgSender) external;
```

```solidity
function prank(address msgSender, bool delegateCall) external;
```

```solidity
function prank(address msgSender, address txOrigin) external;
```

```solidity
function prank(address msgSender, address txOrigin, bool delegateCall) external;
```

### 설명

- `prank(address msgSender)`: **다음 호출에 대해** `msg.sender`를 지정된 주소로 설정합니다. "다음 호출"에는 정적 호출(static call)도 포함되지만, 위임 호출(delegate call)이나 치트 코드 주소에 대한 호출은 포함되지 않습니다.
- `prank(address msgSender, address txOrigin)`: **다음 호출에 대해** `msg.sender`와 `tx.origin` 주소를 설정합니다.
- `prank(address msgSender, bool delegateCall)`: `delegateCall` 값이 `true`이면 **다음 위임 호출에 대해** `msg.sender`를 지정된 주소로 설정합니다.
- `prank(address msgSender, address txOrigin, bool delegateCall)`: `delegateCall` 값이 `true`이면 **다음 위임 호출에 대해** `msg.sender`와 `tx.origin` 주소를 설정합니다.

> ℹ️ **참고**
>
> 위임 호출은 EOA(외부 소유 계정)에서 프랭크(prank)할 수 없습니다.

### 예제

```solidity
/// function withdraw() public {
///     require(msg.sender == owner);

vm.prank(owner);
myContract.withdraw(); // [PASS]
```
