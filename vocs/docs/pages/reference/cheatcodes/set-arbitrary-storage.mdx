## `setArbitraryStorage`

### 시그니처

```solidity
function setArbitraryStorage(address target) external;
```

### 설명

주어진 주소의 스토리지를 완전히 상징적(symbolic)으로 만드는 유틸리티 치트코드입니다.
대상 스토리지에 대한 후속 `SLOAD`는 임의의 값을 읽으며, 동일한 슬롯이 다시 로드되면 이 값이 기억되어 반환됩니다.
스토리지 슬롯이 명시적으로 기록되면(첫 번째 로드 전이나 후에), 기록된 값이 반환됩니다.

### 예제

다음과 같은 스토리지 레이아웃을 가진 컨트랙트의 경우:
```solidity
contract Counter {
    address[] public owners;

    function getOwner(uint256 pos) public view returns (address) {
        return owners[pos];
    }

    function setOwner(uint256 pos, address owner) public {
        owners[pos] = owner;
    }
}
```
`setArbitraryStorage` 치트코드를 사용하면 임의의 값이 반환되도록 보장합니다:
```solidity
contract ArbitraryStorageTest is Test {
    function testArbitraryStorage() public {
        Counter counter = new Counter();
        vm.setArbitraryStorage(address(counter));
        // 임의의 스토리지가 없으면 다음 호출은 배열 범위를 벗어나 실패합니다.
        address owner = counter.getOwner(55);
        // 동일한 슬롯에 대한 후속 호출은 동일한 값을 반환합니다.
        assertEq(counter.getOwner(55), owner);
        // 명시적으로 기록된 경우 새 값이 반환됩니다.
        counter.setOwner(55, address(111));
        assertEq(counter.getOwner(55), address(111));
    }
}
```
