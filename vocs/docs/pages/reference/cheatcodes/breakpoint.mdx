## `breakpoint`

### 서명

```solidity
function breakpoint(string) external;
function breakpoint(string, bool) external;
```

### 설명

디버거 뷰에서 점프할 중단점(breakpoint)을 배치합니다.

`vm.breakpoint('<char>, true)`를 호출하는 것은 `vm.breakpoint('<char>)`와 동일하지만, `vm.breakpoint('<char>, false)`를 호출하면 `'<char>`의 중단점을 지웁니다.

문자가 덮어써지면 실행 단계에서 마지막으로 방문한 것만 고려됩니다.

### 예제

```solidity
function testBreakpoint() public {
    vm.breakpoint("a");
}
```

테스트 환경에서 디버거를 열고 `'a`를 누르면 중단점 치트코드가 호출된 위치로 디버거 단계가 이동합니다.
