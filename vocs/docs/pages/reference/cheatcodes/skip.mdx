## `skip`

## 시그니처

```solidity
function skip(bool skip) external;
```

## 설명

테스트를 조건부로 건너뛴 것으로 표시합니다. 테스트가 실행되지 않고 건너뛰어지도록 하려면 테스트 맨 위에서 호출해야 합니다.

`skip`이 `false` 불리언 값으로 호출되면 테스트를 건너뛰지 않습니다.

건너뛴 것으로 표시된 테스트는 테스트 러너와 요약에 `[SKIPPED]` 라벨과 함께 표시되어 건너뛴 테스트를 쉽게 식별할 수 있습니다.

### 예제

```solidity

function testSkip() public {
    vm.skip(true);
    /// 이 리버트(revert)는 테스트가 건너뛰어지므로 도달하지 않습니다.
    revert("Should not reach this revert");
}

function testNotSkip() public {
    vm.skip(false);
    /// 이 리버트(revert)는 테스트가 건너뛰어지지 않으므로 도달하고, 테스트는 실패합니다.
    revert("Should reach this revert");
}
```
