## `startBroadcast`

### 시그니처

```solidity
function startBroadcast() external;
```

```solidity
function startBroadcast(address who) external;
```

```solidity
function startBroadcast(uint256 privateKey) external;
```

### 설명

테스트 컨트랙트를 호출하는 주소 또는 제공된 주소/개인 키를 발신자로 사용하여, 이후의 모든 호출(현재 호출 깊이에서만, 치트코드 호출 제외)이 나중에 서명되어 온체인으로 전송될 수 있는 트랜잭션을 생성하도록 합니다.

### 예제

```solidity
function t(uint256 a) public returns (uint256) {
    uint256 b = 0;
    emit log_string("here");
    return b;
}

function deployOther() public {
    vm.startBroadcast(ACCOUNT_A);
    Test test = new Test();

    // 트랜잭션을 트리거합니다
    test.t(1);

    vm.stopBroadcast();

    // 다시 브로드캐스트합니다. 이번에는 환경 변수의 개인 키를 사용합니다.
    vm.startBroadcast(vm.envUint("PRIVATE_KEY"));
    test.t(3);
    vm.stopBroadcast();
}
```
