## `difficulty`

### 서명

```solidity
function difficulty(uint256) external;
```

### 설명

`block.difficulty`를 설정합니다.

병합 후(Post-merge) EVM 버전(Paris 및 이후)과 함께 사용하면 리버트됩니다. 그 경우 대신 [`vm.prevrandao`][prevrandao]를 사용하세요.

### 예제

```solidity
vm.difficulty(25);
emit log_uint(block.difficulty); // 25
```
