## `deriveKey`

### 서명

```solidity
function deriveKey(
  string calldata mnemonic,
  uint32 index
) external returns (uint256);
```

```solidity
function deriveKey(
  string calldata mnemonic,
  string calldata path,
  uint32 index
) external returns (uint256);
```

### 설명

주어진 니모닉(mnemonic) 또는 니모닉 파일 경로에서 개인 키를 파생합니다.

첫 번째 서명은 파생 경로 `m/44'/60'/0'/0/{index}`를 사용합니다.
두 번째 서명을 사용하면 두 번째 매개변수로 파생 경로를 지정할 수 있습니다.

### 예제

테스트 니모닉의 `m/44'/60'/0'/0/0` 경로에서 개인 키 파생:

```solidity
string memory mnemonic = "test test test test test test test test test test test junk";
uint256 privateKey = vm.deriveKey(mnemonic, 0);
```

테스트 니모닉의 `m/44'/60'/0'/1/0` 경로에서 개인 키 파생:

```solidity
string memory mnemonic = "test test test test test test test test test test test junk";
uint256 privateKey = vm.deriveKey(mnemonic, "m/44'/60'/0'/1/", 0);
```
