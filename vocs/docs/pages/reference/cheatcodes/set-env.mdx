## `setEnv`

### 시그니처

```solidity
function setEnv(string calldata key, string calldata value) external;
```

### 설명

환경 변수 `key=value`를 설정합니다.

> ℹ️ **참고**
>
> 프로세스에 의해 설정된 환경 변수는 해당 프로세스와 자식 프로세스에서만 액세스할 수 있습니다. 따라서 `setEnv`를 호출하면 현재 실행 중인 `forge` 프로세스의 환경 변수만 수정되며 셸(`forge`의 부모 프로세스)에는 영향을 미치지 않습니다. 즉, `forge` 프로세스가 종료된 후에는 유지되지 않습니다.

### 팁

- 환경 변수 키는 비어 있을 수 없습니다.
- 환경 변수 키에는 등호 `=` 또는 NUL 문자 `\0`이 포함될 수 없습니다.
- 환경 변수 값에는 NUL 문자 `\0`이 포함될 수 없습니다.

### 예제

```solidity
string memory key = "hello";
string memory val = "world";
cheats.setEnv(key, val);
```
