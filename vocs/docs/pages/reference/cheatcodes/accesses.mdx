## `accesses`

### 서명

```solidity
function accesses(
  address
)
external
returns (
  bytes32[] memory reads,
  bytes32[] memory writes
);
```

### 설명

주소에서 읽은(`reads`) 또는 쓴(`writes`) 모든 스토리지 슬롯을 가져옵니다.

먼저 [`record`](/reference/cheatcodes/record.mdx)가 호출되어야 합니다.

> ℹ️ **참고**
>
> 모든 쓰기는 추가적인 읽기로도 간주됩니다.

### 예제

```solidity
/// contract NumsContract {
///     uint256 public num1 = 100; // slot 0
///     uint256 public num2 = 200; // slot 1
/// }

vm.record();
numsContract.num2();
(bytes32[] memory reads, bytes32[] memory writes) = vm.accesses(
  address(numsContract)
);
emit log_uint(uint256(reads[0])); // 1
```
