## `rememberKey`

### 시그니처

```solidity
function rememberKey(uint256 privateKey) external returns (address);
```

### 설명

개인 키를 forge의 로컬 지갑에 저장하고 해당 주소를 반환합니다. 이 주소는 나중에 [브로드캐스팅](/reference/cheatcodes/broadcast.mdx)에 사용할 수 있습니다.

### 예제

테스트 니모닉 `m/44'/60'/0'/0/0` 경로에서 개인 키를 파생시키고, forge의 지갑에 저장한 다음 트랜잭션 브로드캐스팅에 사용합니다:

```solidity
string memory mnemonic = "test test test test test test test test test test test junk";
uint256 privateKey = vm.deriveKey(mnemonic, 0);
address deployer = vm.rememberKey(privateKey);

vm.startBroadcast(deployer);
...
vm.stopBroadcast();
```

`PRIVATE_KEY` 환경 변수에서 개인 키를 로드하고 트랜잭션 브로드캐스팅에 사용합니다:

```solidity
address deployer = vm.rememberKey(vm.envUint("PRIVATE_KEY"));

vm.startBroadcast(deployer);
...
vm.stopBroadcast();
```
