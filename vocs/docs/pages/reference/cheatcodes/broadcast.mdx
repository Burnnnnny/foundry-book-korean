## `broadcast`

### 서명

```solidity
function broadcast() external;
```

```solidity
function broadcast(address who) external;
```

```solidity
function broadcast(uint256 privateKey) external;
```

### 설명

테스트 컨트랙트를 호출하는 주소 또는 발신자로 제공된 주소 / 개인 키를 사용하여, 다음 호출(이 호출 깊이에서만, 치트코드 호출 제외)이 나중에 서명되어 온체인으로 전송될 수 있는 트랜잭션을 생성하게 합니다.

### 예제

```solidity
function deploy() public {
    vm.broadcast(ACCOUNT_A);
    Test test = new Test();

    // 이것은 서명할 tx를 생성하지 않습니다
    uint256 b = test.t(4);

    // 이것은 생성합니다
    vm.broadcast(ACCOUNT_B);
    test.t(2);

    // 환경 변수의 개인 키를 사용하여 이것도 생성합니다
    vm.broadcast(vm.envUint("PRIVATE_KEY"));
    test.t(3);
}
```
