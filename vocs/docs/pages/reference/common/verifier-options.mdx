`--verifier` *name*   
&nbsp;&nbsp;&nbsp;&nbsp;검증 제공자입니다. 사용 가능한 옵션: `etherscan`, `sourcify` & `blockscout`. 기본값: `etherscan`. 참고: Blockscout 홈페이지 탐색기 URL 끝에 "/api\?"를 추가해야 합니다.

`--verifier-url` *url*  
&nbsp;&nbsp;&nbsp;&nbsp;검증 요청을 제출하기 위한 선택적 검증자 URL입니다.  
&nbsp;&nbsp;&nbsp;&nbsp;환경 변수: `VERIFIER_URL`
