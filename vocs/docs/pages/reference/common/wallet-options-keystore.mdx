#### 지갑 옵션 - 키스토어 (Wallet Options - Keystore)

`--keystore` _path_  
&nbsp;&nbsp;&nbsp;&nbsp;주어진 폴더 또는 파일의 키스토어를 사용합니다.  
&nbsp;&nbsp;&nbsp;&nbsp;환경 변수: `ETH_KEYSTORE`

`--account` _account-name_  
&nbsp;&nbsp;&nbsp;&nbsp;파일 이름을 통해 기본 키스토어 폴더(~/.foundry/keystores)의 키스토어를 사용합니다.  
&nbsp;&nbsp;&nbsp;&nbsp;환경 변수: `ETH_KEYSTORE_ACCOUNT`

`--interactive`

`--password` _password_  
&nbsp;&nbsp;&nbsp;&nbsp;키스토어 비밀번호입니다. `--keystore`와 함께 사용됩니다.
&nbsp;&nbsp;&nbsp;&nbsp;환경 변수: `ETH_PASSWORD`
