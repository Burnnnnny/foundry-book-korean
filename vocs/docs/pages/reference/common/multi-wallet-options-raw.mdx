#### 지갑 옵션 - Raw (Wallet Options - Raw)

`-i`
`--interactives` *num*  
&nbsp;&nbsp;&nbsp;&nbsp;개인 키를 입력할 수 있는 대화형 프롬프트를 엽니다. 입력할 키의 개수를 값으로 받습니다.  
&nbsp;&nbsp;&nbsp;&nbsp;기본값: 0

`--mnemonic-indexes` *indexes*  
&nbsp;&nbsp;&nbsp;&nbsp;주어진 니모닉 인덱스의 개인 키를 사용합니다. --mnemonic-path와 함께 사용됩니다.  
&nbsp;&nbsp;&nbsp;&nbsp;기본값: 0

`--mnemonic-paths` *paths*  
&nbsp;&nbsp;&nbsp;&nbsp;지정된 경로의 니모닉 파일을 사용합니다.

`--private-key` *raw_private_key*  
&nbsp;&nbsp;&nbsp;&nbsp;제공된 개인 키를 사용합니다.

`--private-keys` *raw_private_keys*  
&nbsp;&nbsp;&nbsp;&nbsp;제공된 개인 키들을 사용합니다.
