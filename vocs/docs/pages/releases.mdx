# v1.3.1 [2025년 8월 12일]

# Foundry v1.3.1

Foundry v1.3.1은 anvil, cast, forge를 위한 버그 수정 릴리스입니다.

## Anvil 수정 사항
- fix(anvil): eth/backend/mem/mod.rs에서 unwrap 패닉 수정 ([#11141](https://github.com/foundry-rs/foundry/pull/11141)) by [@zerosnacks](https://github.com/zerosnacks)
- fix: 빈 7702 auths 사용 ([#11182](https://github.com/foundry-rs/foundry/pull/11182)) by [@mattsse](https://github.com/mattsse)

## Cast 수정 사항
- fix(cast): 로컬 아티팩트 사용 시 모든 공급자 사용 ([#11090](https://github.com/foundry-rs/foundry/pull/11090)) by [@grandizzy](https://github.com/grandizzy)
- fix(cast): 런타임 검증을 통한 mktx --raw-unsigned 처리 개선 ([#11111](https://github.com/foundry-rs/foundry/pull/11111)) by [@mablr](https://github.com/mablr)

## Forge 수정 사항
- fix(forge): 해결되지 않은 import 일관된 처리 ([#11164](https://github.com/foundry-rs/foundry/pull/11164)) by [@grandizzy](https://github.com/grandizzy)
- fix(lint): 다중 lint 패스에서 비활성화된 id 제외 ([#11122](https://github.com/foundry-rs/foundry/pull/11122)) by [@0xrusowsky](https://github.com/0xrusowsky)
- fix(forge): 리매핑 sol 파일 수정을 위해 컴파일러 버전 올림 ([#11184](https://github.com/foundry-rs/foundry/pull/11184)) by [@grandizzy](https://github.com/grandizzy)
- fix(coverage): 중복된 함수 이름 명확화 ([#11188](https://github.com/foundry-rs/foundry/pull/11188)) by [@DaniPopes](https://github.com/DaniPopes)
- fix(forge): 활성 포크에 상태 변경 저널 설정 ([#11196](https://github.com/foundry-rs/foundry/pull/11196)) by [@grandizzy](https://github.com/grandizzy)
- fix(forge): etherscan 검증 실패에 대한 오류 메시지 개선 ([#11194](https://github.com/foundry-rs/foundry/pull/11194)) by [@shiyasmohd](https://github.com/shiyasmohd)
- fix(fmt): stdin 읽기 실패 시 패닉 방지 ([#11226](https://github.com/foundry-rs/foundry/pull/11226)) by [@DaniPopes](https://github.com/DaniPopes)
- fix(forge script): 동일한 RPC로 반복된 vm.createSelectFork가 segfault 유발하는 문제 수정 ([#11250](https://github.com/foundry-rs/foundry/pull/11250)) by [@zerosnacks](https://github.com/zerosnacks)
