## cast to-hexdata

입력을 소문자, 0x 접두사가 붙은 hex로 정규화합니다.

입력값은 다음이 될 수 있습니다:
- 0x 접두사가 있거나 없는 혼합 대소문자 hex
- ':'로 연결된 0x 접두사가 있는 hex
- 파일의 절대 경로
- 환경 변수에 정의된 태그 (@tag)

```bash
$ cast to-hexdata --help
```
