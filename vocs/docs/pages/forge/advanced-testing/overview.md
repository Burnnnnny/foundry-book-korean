Forge는 여러 고급 테스트 방법을 제공합니다:

- [테이블 테스트 (Table Testing)](/forge/advanced-testing/table-testing)
- [퍼즈 테스트 (Fuzz Testing)](/forge/advanced-testing/fuzz-testing)
- [불변성 테스트 (Invariant Testing)](/forge/advanced-testing/invariant-testing)
- [차등 테스트 (Differential Testing)](/forge/advanced-testing/differential-ffi-testing)

미래에 Forge는 다음도 지원할 예정입니다:

- [심볼릭 실행 (Symbolic Execution)](#)
- [변형 테스트 (Mutation Testing)](#)

각 장에서는 테스트 방법이 해결하는 문제와 이를 자신의 프로젝트에 적용하는 방법에 대해 자세히 설명합니다.
